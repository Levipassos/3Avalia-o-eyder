# Explorador de Países e Culturas

Este projeto é um explorador interativo de países desenvolvido como parte da 3ª Avaliação da disciplina de Programação Web da Universidade Estadual do Piauí - UESPI, ministrada pelo professor Eyder Rios.

A aplicação web permite aos usuários buscar, filtrar, visualizar detalhes e salvar como favoritos informações sobre países de todo o mundo, consumindo dados de uma API pública.

# Como Executar o Projeto
Pré-requisitos:
- Node.js instalado para usar o http-server, ou você pode usar qualquer servidor web local de sua escolha

# Instalação e Execução:
- Clone o repositório ou extraia os arquivos do projeto
- Instale o http-server (npm install -g http-server), ou configure seu servidor para servir as páginas html
- Navegue até a pasta do projeto e execute "http-server -p 8045" se for usar o http-server
- Acesse a página via "http://localhost:8045/index.html"

# Créditos
Este projeto utiliza a [REST Countries API](https://restcountries.com/) para obter dados sobre os países. Agradecemos aos mantenedores desta API por disponibilizar esses dados de forma gratuita.
